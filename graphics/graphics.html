La galerie d'art du projet GNU

Pas de gifs à cause des problèmes de brevets.

A côté de l'art sur les pages web GNU, nous avons, au projet GNU, l'art :

  • venant de l'initiative de Brave GNU World's pour les icônes "We run GNU"
  • sur les couvertures des manuels GNU
  • sur les T-shirts GNU
  • sur les couvertures des bulletins GNU
  • de la collection des badges/pin's GNU. Vous pouvez récupérer les images des pin's depuis un un site FTP GNU dans un fichier appelé button.N.N.shar (où chaque N est un numéro de version à 1, 2 ou 3 chiffres)
