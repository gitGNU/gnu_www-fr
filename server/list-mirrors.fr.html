Miroirs de www.gnu.org

Vous trouverez ici la liste, par pays, des miroirs de www.gnu.org, le principal serveur Web du projet GNU sponsorisé par la Free Software Foundation (FSF). A cet endroit, vous trouverez la liste de nos miroirs FTP.

Si vous souhaitez proposer un site miroir, veuillez lire les conseils, et ensuite demander à webmasters@gnu.org d'ajouter votre votre miroir une fois qu'il est en place.

Le site Web principal du projet GNU est souvent surchargé. Veuillez utiliser le miroir qui est le plus proche de vous.

Ci dessous, la liste des pays ayant un miroir :
